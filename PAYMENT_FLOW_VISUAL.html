📊 Complete Data Flow 

Step 1: User selects Date & Time
└─ Data: { doctorId, doctorName, date, time, fees }

Step 2: Click "Book Appointment"
└─ navigate('/payment', { state: { appointmentData } })

Step 3: Payment Page receives data via location.state
└─ Displays: Patient info, Doctor info, Appointment details, Amount

Step 4: User clicks "Pay Now"
└─ Sends to Backend: {
    name: "AbuTaib",
    email: "abutaib@gmail.com",
    phoneNumber: "9508904653",
    amount: 500,
    doctorId: 1,
    date: "2026-04-15",
    time: "10:00 AM"
  }

Step 5: Backend creates Razorpay Order
└─ Returns: { orderId, paymentLink }

Step 6: Frontend opens Razorpay Checkout
└─ User completes payment in Razorpay modal

Step 7: Payment Success Callback
└─ Backend saves BookingEntity with:
    - patientId, doctorId, date, time, status
    - Sends confirmation email

Step 8: Redirect to Appointments Page
└─ Shows new booking in appointment list

🗄️ Database Updates Needed

PaymentOrderEntity (Updated) 

+ doctorId: Long
+ doctorName: String
+ appointmentDate: String
+ appointmentTime: String
+ patientId: Long
+ specialization: String

BookingEntity (For saving confirmed bookings) 

+ id: Long (PK)
+ patientId: Long (FK)
+ doctorId: Long (FK)
+ appointmentDate: String
+ appointmentTime: String
+ status: String (CONFIRMED, PENDING, CANCELLED)
+ paymentId: String (from Razorpay)
+ fee: Double
+ createdAt: LocalDateTime
